# Allow AVIF Uploads
A WordPress plugin that enables AVIF image uploads and provides comprehensive image management features.

## Features

- AVIF image upload support

- Bulk image conversion

- Lazy loading optimization

- Image metadata handling

- Advanced image optimization

- Queue management for processing

- SEO-friendly image handling

- Server compatibility checking

- WebP fallback support

- Customizable settings

## Core Components

The plugin consists of several core classes that handle different aspects of image management:

- `Bulk_Converter`: Handles batch conversion of images

- `Display`: Manages image display rendering

- `Lazy_Loading`: Implements lazy loading functionality

- `Metadata`: Processes image metadata

- `Optimizer`: Handles image optimization

- `Queue_Manager`: Manages processing queues

- `SEO`: Handles SEO-related image features

- `Server_Check`: Verifies server compatibility

- `Settings`: Manages plugin settings

- `Upload`: Handles image upload processing

- `WebP_Fallback`: Provides WebP fallback support

## Requirements

- WordPress 6.7 or higher

- PHP 8.2 or higher

- GD extension
