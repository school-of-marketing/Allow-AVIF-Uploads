Advanced AVIF image handling with AI optimization, CDN support, and WebAssembly processing.

== Description ==

Allow AVIF Uploads Advanced enables full AVIF support in WordPress with advanced features:

Core Features:
* AVIF upload and conversion support
* Automatic WebP fallback
* Lazy loading optimization
* Bulk conversion tools
* EXIF data management
* SEO optimization
* Animated AVIF support

Advanced Features:
* AI-powered image optimization
* CDN integration
* REST API endpoints
* WebAssembly processing
* Version control system
* Background processing queue

== Installation ==

1. Upload plugin files to `/wp-content/plugins/`
2. Activate through WordPress admin
3. Configure settings under Settings > AVIF Options

== Requirements ==

* PHP 7.4+
* WordPress 5.0+
* Imagick with AVIF support
* 64MB+ PHP memory limit

== Frequently Asked Questions ==

= Does this support animated AVIF? =
Yes, the plugin handles both static and animated AVIF files.

= Can I convert existing images? =
Yes, use the bulk converter under Media > Bulk Convert to AVIF.

= Does it work with CDNs? =
Yes, supports major CDN providers with automatic cache purging.
